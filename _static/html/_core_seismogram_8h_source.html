15 /* A Seismogram is viewed as a special collection of Time Series
16 type data that is essentially a special version of a vector time series.
17 It is "special" as the vector is 3D with real components. One could produce a
18 similar inherited type for an n vector time series object.
19 
20 The structure of a vector time series allows the data to be stored in a matrix.

22 contains core concepts that define a seismogram. It can be extended as in
23 MsPASS to add functionality or aliased to Seismogram to simplify the naming as
24 done, for example, with std::basic_string made equivalent to std::string.
25 */
